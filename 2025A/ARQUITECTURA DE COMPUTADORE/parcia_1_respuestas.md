## Pregunta 24

**Enunciado:**

```
01101010 11001100 10101010
```

Sin comillas, para el texto: "Llámame ;)"  
**Respuesta:**

```
01001100 01101100 11000011 10100001 01101101 01100001 01101101 01100101 00100000 00111011 00101001
```
